保卫 GPA (GPA Defense)

基于 C++ 与 Qt 框架开发的极简 2D 塔防游戏

项目简介

欢迎来到我的项目展示页！《保卫 GPA》是一款诞生于编程课程的硬核塔防游戏。为了保持视觉的统一与纯粹，游戏抛弃了繁冗的伪 3D 渲染，采用了极其干净明了的 2D 矢量美术风格。

在这里，玩家需要合理布局防御塔，抵御一波波试图摧毁你绩点的“DDL 怪物”。

核心技术与特性

  • 🎯 纯粹的 2D 体验： 极其简洁的 UI 设计，清爽的扁平化图标，去除多余的装饰元素。
  • ⚙️ 扎实的底层架构： 纯 C++ 编写，基于 Qt 框架构建 UI 与游戏主循环，并利用 CMake 管理工程构建。
  • 🧠 团队协作： 由“烫烫烫烫队”倾力打造，经历了从需求分析到环境配置调试的完整开发周期。
